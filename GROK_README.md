Transform raw GitHub Copilot Premium data into clear insights that show how effectively your team is using AI to accelerate development. Measure ROI, identify opportunities, and maximize the productivity gains you've already paid for.

**Use instantly online:** Visit [https://pairing4good.github.io/copilot-premium-usage-analyzer/](https://pairing4good.github.io/copilot-premium-usage-analyzer/) - no download required.

## Understanding Premium Requests

GitHub Copilot Premium goes beyond basic code autocomplete. Each developer seat includes 300 premium requests per month that unlock advanced AI features:

**What Premium Requests Enable:**
- Interactive AI pair programming through Copilot Chat
- Complex code generation from natural language descriptions
- Deep codebase analysis and explanations
- Automated code reviews and quality improvements
- Autonomous task completion via AI agents

**What Utilizing These Features Unlocks:**
- **Faster Development**: Reduce time understanding unfamiliar code from hours to minutes
- **Higher Quality**: AI-assisted testing, debugging, and code reviews
- **Rapid Onboarding**: New team members get instant codebase insights
- **Parallel Capacity**: AI agents handle routine tasks while developers focus on complex work
- **Competitive Edge**: Teams that master AI-assisted development ship features faster and tackle technical debt more efficiently

Unused premium requests represent wasted investment—your organization is paying for AI capabilities but receiving only basic autocomplete value.

## How to Use the Tool

### Step 1: Export Your Usage Data (Optional)
Navigate to your GitHub organization settings → Billing → Usage reports → Download "Premium Request Usage Report" CSV.

*Note: If you don't have access to usage reports yet, you can still use the tool to calculate your team's AI capacity potential.*

### Step 2: Open the Tool
Simply open `index.html` in any web browser—no installation required.

### Step 3: Upload and Analyze

![Upload Form](screenshots/001-form.png)

- Upload your CSV file (if available)
- Enter your total number of Copilot Premium seat licenses
- Enter your average developer hourly rate (adjust from the $100 default to match your organization's costs)
- If no CSV is available, check "No premium request analytics CSV available" to see capacity potential
- Click "Analyze Report"

### What You'll See

**Executive Summary Dashboard**

![Executive Dashboard](screenshots/002-exec.png)

- **AI Usage Overview**: Total premium requests used, adoption rate, and utilization percentage
- **AI Developer Capacity**: Total available AI hours, hours actually used, and unused potential
- **Dollar Value**: ROI metrics based on your specified developer hourly rate

**Detailed Analysis**

![Detailed Report](screenshots/003-detail.png)

- **Key Insights**: AI-generated recommendations for improving adoption and utilization
- **Usage Trends**: Daily patterns and top performers
- **Model Distribution**: Which AI models your team prefers
- **User Details**: Complete breakdown by developer

## Privacy & Security

This tool runs entirely in your browser. No data is sent to any server—your usage reports remain completely private and never leave your machine. Safe to use with sensitive organizational data.
