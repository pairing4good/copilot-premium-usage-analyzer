# Source Files

This directory contains all the application source code and assets that are loaded by `index.html` at the root.

## Files

- **app.js** - Main JavaScript application logic
  - CSV data parsing and analysis
  - Chart rendering with Chart.js
  - Form validation and user interactions
  - Productivity calculations and metrics
  
- **styles.css** - Complete stylesheet
  - Responsive design (mobile, tablet, desktop)
  - Grid layouts and animations
  - Print-friendly styles
  
- **favicon.svg** - Application icon

## Usage

The root `index.html` references these files:
```html
<link rel="icon" type="image/svg+xml" href="src/favicon.svg">
<link rel="stylesheet" href="src/styles.css">
<script src="src/app.js"></script>
```
