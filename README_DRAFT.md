# README Draft

## Overview
This project analyzes usage data for premium Copilot users to improve their coding experience.

## Features
- Usage analytics for various Copilot features
- Visual reports on user engagement
- Suggestions for better usage practices

## Installation
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Start the application with `npm start`.

## Usage
After installation, access the tool via your browser to view analytics.
